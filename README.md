# MITAOE PYQs

Enhanced interface for accessing MITAOE's previous year question papers.

## Quick Start

```bash
# Install dependencies
npm install

# Set up environment
cp .env.example .env
# Add your MongoDB URI to .env

# Run development server
npm run dev
```
